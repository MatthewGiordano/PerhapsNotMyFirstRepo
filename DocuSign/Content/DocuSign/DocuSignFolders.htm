DocuSign Folders

DocuSign allows you to better organize your content using Folders.

 

There are 2 types:

  • Folders - For content that only you can access

  • Shared Folders - For content that you want to allow others to access

 

 

Folders can be used on the Manage and Templates tabs.

 

Shared Folders can be used on the Templates tab

 

NOTE: Shared Folders can be used by anyone that has access to them, but can only be created by a DocuSign Administrator.

 

 

Creating a Personal Folder in DocuSign

 

  1. Navigate to the "Templates Tab"

     

    NOTE: Only you can view the content in your personal folder.

     

  2. Click the "+" symbol next to "FOLDERS"

     

  3. Enter a name for the folder in the pop-up dialogue box

  4. Click "CREATE"

     

    NOTE: The newly createdfolder will appear under "FOLDERS"

     

  5. Click the three dots next to the folder name to:

    • Rename the folder

    • Delete the folder

    • Move the folder

    • Create a sub-folder

 

 

Moving Items Into a Folder

To move items into a folder:

  1. Select the item using the checkbox

  2. Click EITHER the "MOVE" button above OR

    • Click "Move" from the "USE" dropdown

     

  3. Wait for the "Move to Folder" dialogue box to appear

  4. View the available folders

     

    NOTE: New folders can be created in this window by selecting "NEW FOLDER"

     

     

  5. Select the desired folder

  6. Click "MOVE"

     

    NOTE: Folders can also be dragged and dropped into the desired folder

     

     

  7. Check to ensure the item is now located within the desired folder
