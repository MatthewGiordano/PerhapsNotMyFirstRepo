DocuSign Reports Tab

The Reports tab allows you to run preconfigured reports related to DocuSign activity.

  1. Select "ALL" to display all available reports

  2. Click the "VIEW" button to the right of each report to view each report

  3. Click the drop-down to schedule when the report should run

     

  4. Select "Envelope from the left-hand menu to display reports related to Envelope activity

     

  5. Select "Recipient" from the left-hand menu to display reports related to the activity of individuals you have send documents to

     

    NOTE: The "Custom Reports" functionality has not yet been implemented at Martin Marietta
