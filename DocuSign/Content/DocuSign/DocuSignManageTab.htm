DocuSign Manage Tab

 

The Manage tab will show an overview of your current DocuSign items.

 

The box on the left allows you to view your document items based on their current Envelope status.

 

 

Inbox Tab

To use the Inbox tab:

  1. Select "Inbox" from the left-hand menu to display items sent to you

  2. Click the "DOWNLOAD" drop-down on the right-hand side to

    • Print items

    • Move items

    • Forward items

    • Delete items

    • Item history

 

 

Sent Tab

To use the sent tab:

  1. Select "Sent" from the left-hand menu to display items sent needing a signature

  2. Sort or filter the list using the box in the upper-right corner

 

 

Drafts Tab

To use the Draft tab:

  1. Select "Drafts" from the left-hand menu to display items in progress that have not yet been completed

  2. Sort or filter by using the "FILTERS" box in the upper-right corner

 

 

Deleted Tab

To use the Deleted tab:

  1. Select "Deleted" from the left-hand menu to display items deleted within the last 6 months

 

 

Quick Views Section

To use the Quick Views section:

  1. Navigate to the "Quick Views" menu from the left-hand menu to display items:

    • Requiring action

    • Awaiting action by others

    • Expiring soon

    • Completed

    • Failed authentication
