Introduction To DocuSign

 

DocuSign is a web based application that allows users to send and sign documents electronically.  The documents are stored in the cloud and can be accessed anytime from anywhere.

 

When sending documents for signature users can control who needs to sign and who can observe.

In addition if there are multiple viewers or signers, users can control the order the document will be signed in.

 

Documents are sent for signature using an electronic Envelope.

 

DocuSign Envelopes are similar to a paper envelope since it is used to "transmit" documents to a recipient.  The key difference is that no physical media is needed since DocuSign transmits documents electronically.

 

Note that an Envelope may contain one or more documents and can be sent to multiple recipients.

 

 

DocuSign users fall into 3 groups:

 

  • Senders: Users enabled to create Envelopes and send documents.

  • Recipients: Users that receive Envelopes and documents for viewing or signing.

  • DocuSign Administrators: Users enabled to perform higher level functions in DocuSign (such as creating Shared Folders).
