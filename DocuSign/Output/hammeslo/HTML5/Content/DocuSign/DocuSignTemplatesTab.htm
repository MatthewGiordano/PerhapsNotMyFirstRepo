DocuSign Templates Tab

 

Templates may be used when you routinely send the same, or similar, Envelopes, even though they may contain different documents, or different recipients each time.

 

DocuSign lets you create a Template from an existing Envelope or create one from scratch.

 

Both methods will be covered here, as well as instructions on how to use a Template once you have created it.

 

Much of the process for creating a Template is similar to Sending a Document (Opens a new tab).

 

 

Save an Existing Envelope as a Template

 

  1. Select the "Templates" tab

     

    NOTE: DocuSign may display a recently sent envelope and give the option for you to save it as a template.

     

    NOTE: Templates can also be created from an existing Envelope by selecting the "Manage" tab from the top menu.

     

  2. Select "Completed" on the left-side menu to show your completed Envelopes

  3. Click the Envelope needed to create the template

     

  4. Wait for the record of the Envelope to open

  5. Click on the "More" drop-down

  6. Select "Save as Template"

     

  7. Wait for the Template screen to open

  8. Verify that it contains the document that was in the original Envelope

  9. Enter the name and an optional description of the template

     

    NOTE: Documents can be added or removed as needed.

     

     

  10. Enter the required recipient information in the "Add Recipient" section

  11. Ensure the "Set signing order" box is checked when there are multiple recipients

  12. Ensure the proper signing order is set

  13. Add an optional subject line and message that will appear in the email notification sent to recipients along side the Envelope

  14. Complete any other desired modifications

  15. Click the "NEXT" button

     

    NOTE: The document will open and display the Initials, Signatures, and text fields that were in the original document. These can be modified if needed. Following any changes, be sure to save by clicking the "SAVE AND CLOSE" button.

 

 

Create a New Template From Scratch

To create a new template from scratch:

  1. Select the "Templates" tab

  2. Click the "NEW" drop-down

  3. Select "CREATE A TEMPLATE"

     

  4. Wait for the "Templates" screen to open

  5. Assign a name and optional description to the Template

  6. Click the "UPLOAD" button to add a document to the Template

     

  7. Verify that the document is not attached to the "Template"

     

  8. Add recipients as needed

  9. Use the "Message to All Recipients" section to add a subject line and a message that will appear within the email notification

     

  10. Click "NEXT" when done

  11. Add the desired fields to the appropriate locations by dragging and dropping them from the left task bar

     

    NOTE: Fields include initials, signatures, data entry, etc.

     

  12. Click the "RECIPIENT PREVIEW" to preview what the recipients will see after launching the document

     

  13. Close the preview by clicking the "X" in the upper right corner of the screen

  14. Click "SAVE AND CLOSE"

  15. Find the Template in the future by navigating to "My Templates"

     

  16. Complete all fields in the selected or established template

 

 

Using Templates

To use Templates:

  1. Navigate to the "Templates" tab

  2. Select "My Templates" from the left-hand menu

  3. Click the "USE" button on the right to use a Template

     

  4. Add any information as needed on the next screen

  5. Make changes by clicking the "ADVANCED EDIT" button

     

  6. Click "SEND" to send the Envelope when done

     

    NOTE: If you clicked "ADVANCED EDIT" then you will be taken back to steps for editing the Template.
